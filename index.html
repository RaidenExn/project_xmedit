Project XMEdit

A desktop application for manually editing specific DHPO XML medical resubmission claim files, designed to streamline the claim correction process.

Download Latest Version

Features

Comprehensive Editing

Modify claim details, totals, activities, and resubmission data with a user-friendly interface.

Diagnosis Management

Add/delete diagnoses with a built-in ICD-10 code and description search.

Observation Manager

Full CRUD support for observations, including file attachments with drag-and-drop.

Bulk Operations

Merge text-based observations and transfer observations from deleted activities.

Customizable UI

Supports both Dark and Light modes with multiple theme colors to fit your workflow.

Installation

1. Install Required Runtime

This application requires the Microsoft Visual C++ Redistributable. Please download and install the latest X64 version from the official Microsoft website.

Download Visual C++ Redistributable

2. Unzip and Run

  1. Unzip the downloaded release file from the Releases Page.
  2. Open the folder and run project_xmedit.exe.
